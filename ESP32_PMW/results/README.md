# Experimental results

- `single_vs_multi_coil/` — single-coil vs multi-coil current profile,
  generated by `tools/plot_coil_current_profiles.py` from
  `data/2026-07-04a_coupling-matrix/coupling_pairwise_20260704_143320.csv`.
- `pid_decoupling_tuning/` — feedforward coil-decoupling compensation tuning
  (baseline / tune 1 / tune 2 ...), generated by
  `tools/plot_pid_tuning_matrix.py` from
  `data/2026-07-04e_recalibration-ccw/` (CCW, 3-stage) and
  `data/2026-07-04b_comp-calibration-cw/` (CW, 2-stage).

Re-run either script to regenerate the PNGs after any data or styling change.
